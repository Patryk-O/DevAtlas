# DevAtlas Product Specification

## Purpose
An interactive platform helping developers learn technologies and prepare for technical interviews.

## MVP Modules
- Dashboard
- Knowledge Atlas
- Topic Viewer
- Flashcards
- Quiz
- Interview Mode
- Progress
- Settings

## Topic Screen
Each topic contains:
- explanation
- diagrams
- code examples
- interview questions
- common mistakes
- related topics
- estimated learning time
- interview importance
